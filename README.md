# Registry

A simple shadcn registry for my projects.

## Usage

```bash
npx shadcn@latest add https://registry.aayush.page/r/card.json
npx shadcn@latest add https://registry.aayush.page/r/navigation-menu.json
npx shadcn@latest add https://registry.aayush.page/r/animated-link-text.json
npx shadcn@latest add https://registry.aayush.page/r/globals.json
```

## Build

```bash
pnpm registry:build
```
