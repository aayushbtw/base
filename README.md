# Registry

A simple shadcn registry for my projects.

## Usage

### Styles

```bash
pnpm dlx shadcn@latest add https://registry.aayush.page/r/globals.json
```

### UI Components

```bash
pnpm dlx shadcn@latest add https://registry.aayush.page/r/card.json
pnpm dlx shadcn@latest add https://registry.aayush.page/r/navigation-menu.json
pnpm dlx shadcn@latest add https://registry.aayush.page/r/animated-link-text.json
```

### Blocks

```bash
pnpm dlx shadcn@latest add https://registry.aayush.page/r/navbar.json
```

## Build

```bash
pnpm registry:build
```
